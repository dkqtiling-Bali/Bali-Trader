BALI TWO-CPU BRIDGE PREP V012I

This is local JSON structure only. It does not open network ports, fetch internet code,
run remote commands, touch API keys, or enable live orders.

Roles:
- BALI_CPU: local Bali dashboard machine.
- STARGATE_CPU: second CPU that will later submit SIM/PAPER result JSON.
- REFEREE_CPU: optional governor/referee view that compares both bots.

Future V012I+ should define the actual import/export rules.
