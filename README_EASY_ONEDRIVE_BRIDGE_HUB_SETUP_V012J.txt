BALI V012J EASY ONEDRIVE BRIDGE HUB SETUP

Adds a Bali-side OneDrive bridge setup action so you do not need Windows network sharing.

Use after install:
1. Open dashboard > Game Arena / Hub.
2. Click Auto Setup OneDrive Bridge.
3. Bali writes the active bridge config and a path file.
4. Stargate uses the same synced OneDrive/BaliArenaBridge folder.

Safety: LIVE_ORDERS_OFF, CHAMPION_LOCK_LOCKED, NO_API_KEYS, SIM_ONLY, JSON_ONLY_NO_REMOTE_COMMANDS.
