# Bali OS V6B Control Dashboard Guide

V6B adds a local URL dashboard and fixes the V6A safety scan false block.

Recommended operating flow:

1. Open `BALI_START_HERE.bat`
2. Choose `Open Local URL Control Dashboard V6B`
3. Click `START DAY / AUTO SESSION`
4. Review latest recommendation
5. Click `END DAY / GIT SAFE SAVE`
6. Use New Chat Handover when moving to another chat

The dashboard is SAFE_TOOLING_ONLY and does not edit trading logic.
