# Bali OS V6E URL Dashboard Guide

Open `BALI_START_HERE.bat`, then choose `1. Open Local URL Dashboard V6E`.

PC URL:

`http://localhost:8787`

Phone URL may work on the same Wi-Fi if Windows Firewall allows it:

`http://YOUR-PC-IP:8787`

Controls are safe-action only. There is no arbitrary command box, no live trading button, no API key entry, and no champion unlock.

If a button opens a command window, leave it open until it says finished. Use the Latest Log / Latest Safety buttons to inspect results.
