# Bali OS V7 Dashboard Guide

V7 makes the URL dashboard the main daily control surface.

Use local mode on the PC:
- Open `BALI_START_HERE.bat`
- Choose `Open Local URL Dashboard V7`
- Browser URL: `http://localhost:8787`

Use phone/LAN mode:
- Open `BALI_START_HERE.bat`
- Choose `Open Phone/LAN Dashboard V7`
- Use the phone URL shown in the terminal, usually `http://YOUR-PC-IP:8787`
- Phone must be on the same Wi-Fi
- Windows Firewall may need to allow PowerShell/private network access

Safe dashboard actions:
- START DAY / AUTO SESSION
- END DAY / GIT SAFE SAVE
- SAFETY SCAN
- RECOMMEND NEXT PATCH
- NEW CHAT HANDOVER
- View latest status, recommendation, handover, safety, logs, evidence index, and project map

Blocked by design:
- Live trading
- API keys
- Champion unlock
- Arbitrary command execution
- Profitability claims without evidence
