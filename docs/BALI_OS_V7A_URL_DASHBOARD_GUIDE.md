# BALI OS V7A URL Dashboard Guide

V7A makes the dashboard launcher behave like a small local app.

## Daily workflow

1. Open `BALI_START_HERE.bat`.
2. Choose `1. Start Local URL Dashboard V7A`.
3. Leave the PowerShell dashboard server window open.
4. Use the browser dashboard buttons.
5. Use `END DAY / GIT SAFE SAVE` before closing.

## Troubleshooting

If the browser says `localhost refused to connect`, the dashboard server is not running. Reopen `BALI_START_HERE.bat` and choose option 1 again.

If port 8787 is unavailable, V7A tries fallback ports from 8788 to 8792 and prints the working URL.

Phone mode is local Wi-Fi/LAN only and may require Windows Firewall approval. Phone buttons are limited to safe Bali OS actions.
